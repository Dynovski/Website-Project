Quizer Android Application

My first attempt to create Android application!

Quizer is simple testing application. Both teachers and students can use it. They have different functionalities:
- Teachers can login and create tests.
- Students can subscribe to courses and solve tests created for given course.
Project makes use of Firebase in user authentication and preserving application data. I also tried using latest Android Components while making Quizer.

Check project on Github!

Testing Application

My first Java project for event-driven programming.

This project can be treated as a prototype of Quizer. The idea was to create a testing system for both students and teachers. Tests are in the form of simple choice questions with answers, created by teachers. This is a Desktop application created using JavaFX. That was my first programming project that included database.

Check project on Github!

Assembly Median Filter

Median filter for .bmp files for x86 64-bit processors.

That was the project for Computer Architecture course to get familiar with low-level programming. This program filters images, reducing white noise that can be noticed on them. It taught me how much is happening in the processor while programming applications, because I had to program operations on processor's registers manually. Programming in assembly enables developer to create fast running programs with minimal memory usage.

Check project on Github!
